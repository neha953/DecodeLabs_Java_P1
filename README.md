# DecodeLabs Java Project 1 - Number Guessing Game

## Overview
A console-based Number Guessing Game built in Java as part of the DecodeLabs Industrial Training Kit - Batch 2026.

## How It Works
The program generates a random number between 1 and 100. The user has 7 attempts per round to guess the correct number. After every guess, the program provides High or Low feedback. The game supports multiple rounds and tracks the total score.

## Features
- Random number generation using Java Random class
- 7 attempts per round with attempt counter
- High / Low feedback on every guess
- Multiple rounds with Play Again option
- Score tracking across rounds
- Exception handling for invalid inputs
- Scanner buffer trap fix using nextLine()

## Concepts Covered
- Random class
- While loops and do-while loops
- Conditional statements
- User input handling with Scanner
- Exception handling with try-catch
- Program state management
